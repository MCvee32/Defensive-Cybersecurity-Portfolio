# Part 1 - A5: Discover Cryptographic Implementation Used Online

### Implementation Discovered
TLS (Transport Layer Security)
TLS is used to secure HTTPS connections.
Involves:
- Asymmetric encryption for key exchange and server authetication
- Symmetric encryption for encrypting session data
- Hash functions for message integrity
  
**Online Uses**
- Online banking
- E-commerce
- Email services
- Social media

**Purpose**
TLS secures web traffic by encrypting data transmitted online.
Protects CIA triad by ensuring
  - data cannot be read by unauthorised parties
  - data cannot be modified in transit
  - verifies the identity of the server
