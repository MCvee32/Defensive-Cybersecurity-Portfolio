# Part 1 - A14: Discover 5 AI-Enabled Security Solutions

The following are tools/solutions that leverage AI, machine learning, and GenAI, to enhance and automate cybersecurity defenses.

### Solution 1: AI Security Posture Management (AI-SPM)
AI-SPM tools monitor and assess the security of AI systems. They identify risks such as misconfigurations, data exposure, or insecure AI deployments. Helping organisations maintain a strong security posture

**Evidence**
https://www.paloaltonetworks.com/cyberpedia/ai-security-posture-management-aispm

### Solution 2: Extended Detection and Response (XDR)
XDR platforms integrate AI into multiple security layers to monitor endpoints, networks, and servers. XDR uses AI to detect, analyse, and respond to threats. Automating threat detection and response.

### Solution 3: LLM Security and Data Protection
Focuses on securing LLMs by preventing data leakage, prompt injection, and misuse. AI-based controls monitor inputs and outputs to ensure sensitive data is protected and model behaviour remains safe.

**Evidence**

### Solution 4: Network Detection and Response (NDR)
NDR uses AI to analyse network traffic and detect unusual or malicious behaviour. It identifies threats within a network and helps respond to them quickly

**Evidence**
https://www.sophos.com/en-us/cybersecurity-explained/network-detection-and-response

### Solution 5: AI-Powered SIEM (Security Information and Event Management)
AI-Powered SIEM systems collect and analyse logs from across an organisation. AI is used to improve threat detection by identifying patterns, reducing false positives, and automating incident response.
