# Part 1 - A11: Discover 5 Unique Access Control Devices

### Device 1: Keypads
Requires indivduals to enter the correct PIN code in order to access a restricted area. Prevents unauthorised access of restricted areas from others, as only indivduals who know the code can enter.

### Device 2: Key Card / RFID Reader
Uses a card or fob to unlock doors or secure areas. Only authorised individuals are issued cards or fobs to restricted areas, preventing unauthorised access.

### Device 3: Biometric Scanners
Fingerprints, retina scans, or face IDs are collected from authorised individuals. Individuals wishing to access restricted areas are required to perform a biometric scan. Access is only granted to those authorised to access the restricted area.

### Device 4: Mobile-based Access Control
Uses smartphones as digital credentials via Bluetooth Low Energy (BLE), NFC, or Wi-Fi to unlock doors, replacing traditional cards or keys. Provides enhanced security through encrypted, non-clonable digital keys.

### Device 5: Smartlocks
Locks that can be controlled using smartphone apps or key codes. Allows remote locking and unlocking. Only users with correct keycodes or access through the secure smartphone app can operate the lock, preventing unauthorised access.
