# Part 1 - A7: Discover Cryptography Used In Modern Networks

### Implementation Discovered
SHA-256 (Secure Hash Algorithm 256-bit)
SHA-256 is a cryptopgraphic hash function used in modern network protocols to ensure data integrity

**Used In**
- TLS for handshake integrity
- IPsec for authentication
- Digital Certificates

**Purpose**
SHA-256 generates a fixed 256-bit hash value used to verify that transmitted data has not been modified. Supporting integrity and authentication in modern network communications.

**Evidence**
- SHA-256 used on Instagram's website for message integrity
