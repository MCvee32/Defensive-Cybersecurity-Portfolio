# Part 1 - A10: Discover a Privacy Technique Used Offline

### Privacy Technique Discovered
Data Anonymisation
is a privacy technique used to remove or modify personal information so that individuals cannot be identified from stored datasets.
Types:
- Masking: removing sensitive data fields
- Pseudonyms: replacing identifers with fake identifiers

**Used In**
- Databases
- Statistical Reports
- Datasets

**Purpose**
To protect indivdual privacy by removing identifiable information from stored data (e.g. names, addresses, identification numbers, etc). This allows organisations to analyse or share datasets without exposing personal information.
