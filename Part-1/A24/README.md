# Part 1 - A24: Teach Your Family About A Cybersecurity Topic of Your Choice

### Topic Chosen: Hashing and Blockchain
I chose to give my sister an overview of the concept of hashing, and specifically how it used in blockchain. Keeping it as simple as possible as she does not have a technical background.

I explained hashing as the process of taking an input and transforming into a fixed-length string of characters called a hash. I mentioned how hashes are unique to the input so that small changes to the original input can be detected.

I also discussed its implementation in blockchain to ensure a secure and trustworthy system. As each block contains a hash of the previous block to ensure data cannot be tampered.
