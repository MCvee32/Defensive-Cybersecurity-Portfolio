# Part 1 - A13: Discover 5 Online Security Tools

### Tool 1: Firewalls
Monitor incoming and outgoing network traffic and enforce rules to block malicious or unauthorised connections. Provide system protection from attacks like port scanning, intrusion attempts, or malware spreading.

### Tool 2: Autheticator Apps (MFA)
Generate time-based one-time codes for login verification. Even if a password is stolen, attackers cannot access accounts without the code from the user's device, adding an extra security layer.

### Tool 3: Endpoint Protection
Installed on laptops or servers to monitor, detect, and prevent threats. Ensures that each device on your network meets your security standards, protecting each endpoint against online attacks.

### Tool 4: Password Managers
Securely store encrypted passwords for online accounts. Can generate strong, unique passwords for each account. Protects users from password reuse attacks and phishing attempts.

### Tool 5: Antivirus Software
Scans files, emails, and web traffic in real-time for malware, viruses, and ransomware. Provides live security updates on newly discovered threats, protecting systems while connected to the internet.
