# Part 1 - A6: Discover Cryptographic Implementation Used Offline

### Implementation Discovered
Full Disk Encryption (FDE) used for protecting stored data.

**Used In**
- Laptops
- Desktops
- USB drives
e.g. FileVault (macOS) uses AES encryption to protect stored data on macOS devices.

**Purpose**
- Protects data if a device is stolen or physically tampered with
- Prevents unauthorised access without authentication
- Ensures confidentiality of stored data

**Evidence**
- FileVault on my Mac using FDE to secure data on my disk
