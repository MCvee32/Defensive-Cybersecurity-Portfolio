# Part 1 - A2: Discover Security Concepts in Public Space

## Security Concepts Discovered

### 1. Card Access Systems
**Location/System** Public Buildings (Hospitals, Schools, Libraries)

**How It Works**
- Users are issued physical access cards e.g. RFID
- Card is tapped on readers at doors or restricted areas
- Access is granted/denied based on the user's role and permissions
- All access attempts are logged in a system

**Security Benefits**
- Prevents unauthorised access to restricted areas
- Protects physical IT infrastructure or assets from destruction, damage, or tampering
- Creates a log of who accessed specific locations and when

### 2. Public Wifi Network Segmentation
**Location/System** Public Wifi Networks

**How It Works**
- Public Users connect to Public Wifi
- Staff connect to a secure internal network
- Firewalls restrict communication between public and internal networks

**Security Benefits**
- Databases/servers are on their own protected network
- Reduces risk of attacks to internal systems
- Protects sensitive data from public users

### 3. Logging, Monitoring, and Audit Trails

**Location/System** Authentication Systems / Network Infrastructures / Card Access Systems

**How It Works**
- Login attempts, network traffic, and card access events are logged (who, when, and where)
- Security/IT teams monitor logs for suspicious behaviour
- Logs are kept as evidence

**Security Benefits**
- Early threat detection
- Provides evidence for investigations
- Ensures accountability and compliance
