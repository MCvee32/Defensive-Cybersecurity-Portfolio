# Part 1 - A12: Discover 5 Unique Offline Security Tools

### Tool 1: Disk Encryption Software
Use encryption algorithms to encrypt all data on a storage device. Data can only be read with the correct password or encryption key. Data remains protected even if the device is stolen.
e.g. FileVault on Apple Mac

### Tool 2: Paper Shredders
Mechanically shreds physical documents into small unreadable pieces. Making sensitive information like account numbers, personal infromation, or confidential documents unreadable.

### Tool 3: Safes
Physically secure containers that require a key, combination, PIN, or biometric access. They protect private, valuable, or sensitive materials from theft or tampering. 
e.g. Passport stored in safe

### Tool 4: Malware Analysis Tools
Used to examine suspicious software or files in a controlled offline environment to understand how they work and whether they are malicious. Preventing malware from spreading to the system.
e.g. IDA Pro disassembles software to inspect code behaviour

### Tool 5: Security-Focused Operating Systems
Security-focused operating systems have built-in hardened security features to protect offline computers from threats and also provides tools for forensic analysis.
e.g. Kali Linux comes pre-loaded with over 600 security tools for penetration testing and digital analysis
