# Part 2 - B16: Survey the current state-of-the-art solutions in cybersecurity

## Current State-of-the-art Cybersecurity Solutions for Access Control

### 1. Zero Trust Architecture
No user or device is trusted by default, even inside the network. Every access request is verified  
**Key Features:**  
- Continuous Authentication
- Device and location checks
- Micro-segmentation  
**Why it is State-of-the-Art:**  
Minimises attack surface, protection against lateral threat movement, and secure support for hybrid workforces (which is needed in modern organisations)

### 2. Identity and Access Management (IAM) Systems
A centralised security framework used to manage digital identities and control who has access to systems, resources,and data.  
**Key Features:**  
- Single Sign-On
- Role-Based Access Control
- User lifecycle management  
**Why it is State-of-the-Art:**  
Strengthened security against breaches, enhanced operational efficiency through automated workflows, simplified user experience (SSO), and better regulatory compliance.

### 3. Privileged Access Management (PAM)
Secures, controls, and monitors elevated permissions (admin) to mitigate risks, prevent breaches, and ensure compliance.  
**Key Features:**  
- Just-in-time access. Privileges are granted on demand and dropped after set time
- Session monitoring and recording
- Credential vaulting  
**Why it is State-of-the-Art:**  
Prevents misuse of powerful admin accounts, limits damage if credentials are compromised, and secure remote access.
